=========
nidoodles
=========

*nidoodles* is a set of documents and code fragments that we use to record
thoughts about refactoring, reviews of other packages and any other use that
seems sensible.

To edit these docs, please clone or fork from github::

    git clone git://github.com/nipy/nidoodles.git


Note that the "id" in "nidoodles" is silent.
